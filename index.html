프로젝트 개요

두근두근 플레이리스트는 사용자의 감정을 분석하여 그에 맞는 음악을 추천함으로써,

사용자에게 감정적인 위로와 공감을 제공하는 웹게임을 개발하였습니다.

사용자가 선택한 감정에 따라 추천된 플레이리스트를 제공합니다.

프로젝트 주요 기능

감정 선택값 저장

사용자가 대화 흐름 중 클릭한 버튼의 data-emotion 값을 감정 데이터로 인식하여, 사전 정의된 emotionScores 객체에 각 감정별 점수로 누적 저장합니다.

감정 점수 기반 음악 추천

저장된 감정 점수 중 가장 높은 값을 기준으로 해당 감정에 맞는 플레이리스트 배열 중 하나를 무작위로 추천합니다.

결과 화면 재생 여부 선택

결과 화면 진입 시 alert 창을 띄워 사용자가 음악을 바로 재생할지 여부를 선택할 수 있도록 구성하였습니다.

음악 재생 제어 기능

음악 재생 시, 프로그레스바를 클릭하거나 드래그하여 원하는 재생 위치로 이동할 수 있도록 하여 유동적인 음악 감상이 가능합니다.

로컬스토리지 기반 이어하기 기능

사용자의 이름 및 게임 데이터를 로컬스토리지에 저장하여, 게임 중간에 브라우저를 종료해도 다시 접속 시 이어서 진행할 수 있도록 구현하였습니다.

감정 분석 및 대화 분기 시스템

사용자 선택에 따라 대화 흐름이 분기되는 브랜칭 시스템을 통해, 다양한 감정과 상황에 맞는 시나리오를 제공합니다. 또한 ‘이전/다음’ 버튼을 통해 대화 흐름을 자유롭게 오갈 수 있습니다.

감정 기반 캐릭터 및 배경 변화

대화 진행 중 감정 변화에 따라 캐릭터 이미지가 바뀌고, 상황에 맞춰 배경 또한 전환되어 몰입감 있는 사용자 경험을 제공합니다.

추천 결과 공유 기능

추천 결과 화면을 이미지로 저장하거나 링크로 공유할 수 있어, 다른 사람들과 쉽게 결과를 나눌 수 있습니다.
